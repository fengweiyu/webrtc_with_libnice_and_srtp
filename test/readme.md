1.windows
    1.1.运行
        双击run.bat即可运行
    1.2.编译运行
        修改代码后，(注意需要先安装go环境,然后)
        (如果是第一次修改代码后编译，则需要首先双击firstBuild.bat，再)
        双击buildAndRun.bat即可运行
